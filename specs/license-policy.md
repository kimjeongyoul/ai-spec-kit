# 📜 Open Source License Policy

This project strictly manages open source licenses to avoid legal risks.

## 1. Allowed Licenses (Permissive)
You are FREE to use libraries with the following licenses:
- **MIT**
- **Apache License 2.0**
- **BSD (2-Clause & 3-Clause)**
- **ISC**
- **Unlicense / Public Domain**

## 2. Conditional Licenses (Review Required)
Seek human approval before using these:
- **LGPL** (Only if linked as a dynamic library)
- **MPL (Mozilla Public License)**

## 3. Forbidden Licenses (Copyleft / Risky)
NEVER use libraries or snippets under these licenses:
- **GPL (v2, v3)**
- **AGPL (Affero GPL)**
- **SSPL (Server Side Public License)**
- **Creative Commons (Non-Commercial)**

## 4. AI Agent Duty
- Before suggesting a new package or library, **check its license**.
- If the license is not in the "Allowed" list, you MUST warn the user and suggest an alternative.
- Do not copy code snippets from StackOverflow or GitHub unless you can verify they are under a permissive license.
